# SmartCare

SmartCare 是一个校内小团队协作项目，当前采用多模块结构：

- `frontend/` 前端
- `backend/` 后端（Spring Boot）
- `ai-service/` AI 服务
- `database/` 数据库脚本
- `docs/` 项目文档

## 快速开始

后端运行环境：`JDK 17+`（Spring Boot）。

仓库协作工具（提交规范校验）：

```bash
npm install
npm run prepare
```
