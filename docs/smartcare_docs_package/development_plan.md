# 开发计划

## 第一阶段 - 设计与准备（第1周）
- 确认系统架构
- 配置仓库与开发环境
- 准备测试视频与样例数据

## 第二阶段 - AI 原型（第2-3周）
- 实现视频读取
- 实现姿态识别
- 实现跌倒检测逻辑

## 第三阶段 - 后端服务（第4周）
- 创建 Spring Boot 服务
- 实现事件 API
- 连接 MySQL 并完成基础持久化

## 第四阶段 - 数据库集成（第5周）
- 完成表结构设计
- 实现告警与事件数据存储

## 第五阶段 - Web 管理端（第6周）
- 构建基础管理页面
- 展示告警列表与老人信息

## 第六阶段 - 集成与演示（第7-8周）
- AI 服务与 Spring Boot 联调
- 前端联调 API
- 系统联测与问题修复
- 准备演示材料
