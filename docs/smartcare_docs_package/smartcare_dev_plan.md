# SmartCare 智慧社区养老监护系统

## 项目方案与阶段开发计划（开发人员版）

------------------------------------------------------------------------

# 1. 项目目标

SmartCare 是一个基于 AI 视觉识别的社区养老安全监护系统。

系统通过摄像头实时分析老人行为，自动识别跌倒或异常行为，并将事件发送到后台系统进行记录和展示，从而提升独居老人的安全监护能力。

系统核心目标：

-   自动识别老人跌倒等异常行为
-   实时生成安全事件
-   提供管理平台查看报警信息
-   为社区养老提供智能化解决方案

------------------------------------------------------------------------

# 2. 系统核心功能（MVP范围）

本项目当前阶段只实现最小可行系统（MVP）。

核心功能包括：

1.  AI视觉行为识别（跌倒检测）
2.  事件接收与记录服务
3.  报警记录管理
4.  Web管理界面展示

------------------------------------------------------------------------

# 3. 系统整体架构

系统采用模块化架构设计，共包含四个核心模块：

1.  AI识别模块
2.  事件服务模块
3.  数据存储模块
4.  管理平台模块

系统架构流程：

摄像头 → AI识别模块 → 事件服务 → 数据库 → Web管理平台

------------------------------------------------------------------------

# 4. 模块设计

## 4.1 AI识别模块

职责：

-   接收摄像头视频流
-   识别人体
-   识别姿态
-   判断是否跌倒
-   生成事件数据

输入：

视频流

输出：

行为事件

示例事件数据：

    {
      "elder_id": 1,
      "event_type": "fall",
      "timestamp": "2026-03-07T10:20:00"
    }

技术：

-   Python
-   OpenCV
-   YOLO
-   MediaPipe

------------------------------------------------------------------------

## 4.2 事件服务模块

职责：

-   接收 AI 模块发送的事件
-   校验数据
-   写入数据库
-   提供 API 给前端系统

主要接口：

POST /api/events\
GET /api/alerts\
GET /api/elders

推荐技术：

-   Node.js
-   Express

------------------------------------------------------------------------

## 4.3 数据存储模块

负责系统数据持久化。

数据库存储内容：

-   老人信息
-   报警记录
-   行为事件记录

推荐数据库：

MySQL

主要数据表：

elders\
alerts\
events

------------------------------------------------------------------------

## 4.4 管理平台模块

Web 管理系统提供数据展示功能。

主要功能：

-   老人信息管理
-   报警记录查看
-   实时状态查看
-   数据统计展示

推荐技术：

-   Vue 或
-   简单 HTML + JS

------------------------------------------------------------------------

# 5. 系统数据流

## 正常状态

摄像头 → AI识别 → normal

系统不产生事件。

## 异常状态

摄像头\
↓\
AI识别\
↓\
检测到跌倒\
↓\
发送事件\
↓\
后端接收事件\
↓\
写入数据库\
↓\
前端界面显示报警

------------------------------------------------------------------------

# 6. API接口设计（初步）

POST /api/events

示例数据：

    {
      "elder_id": 1,
      "event_type": "fall",
      "timestamp": "2026-03-07T10:20:00"
    }

------------------------------------------------------------------------

## 获取报警记录

GET /api/alerts

返回示例：

    [
      {
        "id": 1,
        "type": "fall",
        "time": "2026-03-07 10:20"
      }
    ]
