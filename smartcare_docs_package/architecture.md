# 系统架构

## 概述
SmartCare 采用模块化架构，分离了 AI 处理、后端服务、数据库和前端。

## 组件
1. AI 识别服务（Python）
2. 事件服务 API（Node.js / Express）
3. 数据库（MySQL）
4. Web 管理平台

## 架构流程
摄像头 → AI 识别 → 事件 API → 数据库 → Web 仪表板

## 职责

### AI 识别服务
- 视频处理
- 姿态检测
- 跌倒检测
- 事件生成

### 事件 API
- 接收事件
- 验证数据
- 存储警报
- 提供 REST APIs

### 数据库
存储：
- 老人信息
- 警报
- 事件

### Web 平台
显示：
- 老人列表
- 警报历史
- 系统状态
